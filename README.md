# Yunzai_JS_Plugins
轻量型云崽单JS插件仓库

本仓库收集了适用于 TRSS-Yunzai / Miao-Yunzai 等云崽机器人的**单文件即用型插件**，无需安装依赖，下载后放入 `plugins/example/` 目录即可使用。

## 插件列表

| 插件名 | 文件名 | 简介 |
|--------|--------|------|
| 面板图图库管理器 | `面板图图库管理器.js` | 管理喵喵插件角色面板图库（主图库/屏蔽图库），支持状态查看、安装、屏蔽/启用面板图等功能 （已停止维护）|

> 更多插件开发中，欢迎关注更新。

## 面板图图库管理器
> **该单文件版本已停止维护，请使用独立插件包 [ProfileImg-Plugin](https://github.com/AxiuCN/ProfileImg-Plugin) 获取完整功能。**  
> 当前文件仍可正常工作，但不再新增功能或修复。

| 指令 | 权限 | 说明 |
|------|------|------|
| `#图库状态` | 所有人 | 查看主图库与屏蔽图库总览 |
| `#主图库状态` | 所有人 | 查看主图库详细信息 |
| `#屏蔽图库状态` | 所有人 | 查看屏蔽图库详细信息 |
| `#下载主图库` | 主人 | 安装主图库，已安装则提示更新 |
| `#下载屏蔽图库` | 主人 | 安装屏蔽图库，已安装则提示更新 |
| `#<角色名>面板图屏蔽列表` | 所有人 | 查看该角色被屏蔽的面板图 |
| `#屏蔽<角色名>面板图 序号` | 主人 | 将主图库指定图片移入屏蔽图库（支持别名） |
| `#启用<角色名>面板图 序号` | 主人 | 将屏蔽图库指定图片移回主图库（支持别名） |
  
> 主图库地址：[miao-plugin-ProfileImg](https://github.com/AxiuCN/miao-plugin-ProfileImg)  
> 屏蔽图库地址：[miao-plugin-ProfileImg-Blocked](https://github.com/AxiuCN/miao-plugin-ProfileImg-Blocked)
